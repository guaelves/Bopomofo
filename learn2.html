🎵 聲符不可單獨出現

學習上需注意：聲符（子音）不可單獨出現，必須與韻符搭配成完整音節。除了 ㄓ、ㄔ、ㄕ、ㄖ、ㄗ、ㄘ、ㄙ 可單獨成音外，其他聲符都需搭配韻符。

例子：

公雞 → ㄍㄨㄥ ㄐㄧ

油漆 → ㄧㄡˊ ㄑㄧ

小溪 → ㄒㄧˋ

這一頁是注音最基本的規則，提醒孩子「聲符要有韻符才完整」。

🔤 ㄐ、ㄑ、ㄒ → 不能少掉「ㄧ」

這三個聲符在拼讀時，前面常需要搭配介音「ㄧ」。

公雞 → ㄐㄧ

油漆 → ㄑㄧ

小溪 → ㄒㄧ

教孩子記：遇到ㄐ、ㄑ、ㄒ，先想「ㄧ」再念喔！

🐢 ㄍ、ㄎ、ㄏ → 不能少掉「ㄜ」

這三個聲符與韻母結合時，常會出現「ㄜ」音不可省略：

鴿（ㄍㄜ）子

蝌（ㄎㄜ）蚪

喝（ㄏㄜ）水

小朋友常忘記加「ㄜ」，家長可用圖片或動作幫助記憶。

🧸 ㄅㄆㄇㄈ 與 ㄨㄛ 拼讀規則

當 ㄅ、ㄆ、ㄇ、ㄈ 與「ㄨㄛ」拼讀時，不需要把「ㄨ」符號寫出來。

玻璃 → ㄅㄛ ㄌㄧˊ

外婆 → ㄨㄞˋ ㄆㄛˊ

抹布 → ㄇㄛ ㄅㄨˋ

佛祖 → ㄈㄛˊ ㄗㄨˇ

註：是「麼（˙ㄇㄜ）」，不是「麼（˙ㄇㄛ）」。

🔎 ㄥ 的發音（與 ㄨㄥ 的不同）

  • ㄥ 的發音是（ㄜ + ŋ），不是（ㄛ + ŋ）。
  • ㄨㄥ 才是發（ㄛ + ŋ）的音。

燈 → ㄉㄥ（ㄜŋ）

東 → ㄉㄨㄥ（ㄛŋ）

很多孩子會把ㄥ唸成（ㄛŋ），可以用台語「黃」字體會ㄥ的收音。

🔁 聲符與 ㄥ / ㄨㄥ 結合的發音

聲符與 ㄥ 結合時，發（ㄜ + ŋ）的音；聲符與 ㄨㄥ 結合時，發（ㄛ + ŋ）的音。

燈（ㄉㄥ）泡、疼（ㄊㄥˊ）痛 → 屬於 ㄥ 的發音

東（ㄉㄨㄥ）西、同（ㄊㄨㄥˊ）學 → 屬於 ㄨㄥ 的發音

例句與生活單字練習可以幫助孩子分辨。

🟣 ㄅㄆㄇㄈ 與 ㄥ 拼音要注意

ㄅ、ㄆ、ㄇ、ㄈ 與 ㄥ 拼音時，發（ㄜŋ）的音，並且 不能寫出ㄨ。

崩 → ㄅㄥ

朋 → ㄆㄥ

夢 → ㄇㄥ

風 → ㄈㄥ

常見錯誤：寫成「ㄨㄥ」，可透過表格與朗讀矯正。

🔍 ㄨㄥ 的兩種情形（單獨 vs 有聲符）

  • 單獨發音（沒和聲母拼）：ㄨㄥ → 發（ㄨㄛ）音（例：不倒翁、嗡嗡叫的「嗡」）。
  • 有和其他聲符拼時：ㄨㄥ → 發（ㄛ）音（例：通 → ㄊㄨㄥ → 念「ㄊㄛ」）。

仔細聽：翁（單字）和通（有聲符）發音不同，示範對比幫助分辨。

✨ 例外：結合韻 ㄩㄥ 的念法

當 ㄩ 與 ㄥ 結合成「ㄩㄥ」時，實際念法接近「ㄩㄛ」，但寫法仍視為單一韻。

用 → ㄩㄥ（念起來像「ㄩㄛ」）

功 → ㄍㄨㄥ

學生常寫成「ㄩㄨㄥ」，教學時要提醒不要多寫「ㄨ」。

👅 翹舌音與平舌音（容易混淆）

兩組常被混淆的音：

  • ㄓ、ㄔ、ㄕ：翹舌音，舌尖後抵硬口蓋（舌面抬起）。
  • ㄗ、ㄘ、ㄙ：平舌音，舌尖前抵上下牙床間。

練習：翹舌 → ㄓ（ㄓㄨ）、ㄔ（ㄔㄨ）、ㄕ（ㄕㄨ）

平舌 → ㄗ（ㄗㄨ）、ㄘ（ㄘㄨ）、ㄙ（ㄙㄨ）

🔄 常混淆的韻母：ㄛ/ㄡ、ㄝ/ㄟ、ㄜ/ㄦ、ㄣ/ㄥ、ㄢ/ㄤ

  • ㄛ：嘴較圓，嘴型不變。
  • ㄡ：先發ㄛ，再收成 ㄨ 的滑音。
  • ㄝ：嘴型不變，不能和聲符直接拼，中間需有介音 ㄧ 或 ㄩ。
  • ㄟ：先發ㄝ，再收成 ㄧ 的滑音。
  • ㄜ：嘴較扁開，嘴型不變。
    ㄦ：先發ㄜ再捲舌。
  • ㄣ：前鼻音（像 ㄜ + ㄋ）。
  • ㄥ：後鼻音（像 ㄜ + ŋ）。
  • ㄢ / ㄤ：先發ㄚ再收前/後鼻音。

提示：可用台語「黃」字的尾音，體會ㄥ的後鼻音。

⚖️ 常會弄混的子音：ㄈ、ㄏ、ㄌ、ㄋ、ㄖ

  • ㄈ：唇齒音，上排牙齒咬住下嘴唇（如「飛」）。
  • ㄏ：舌根音，嘴唇和牙齒不接觸（如「灰」）。
  • ㄌ：舌尖邊聲帶音，舌尖輕翹滑下帶ㄜ音（如「恐龍」）。
  • ㄋ：舌尖抵住上顎再離開，氣流由鼻腔發出鼻音（如「老農」）。
  • ㄖ：舌尖翹起，接近硬口蓋，氣流擦聲帶音（如「孔融」）。

例：飛機（ㄈㄟㄐㄧ） vs 灰雞（ㄏㄨㄟㄐㄧ）

好樂（ㄌㄜˋ） ≠ 好熱（ㄖㄜˋ） — 小心不要念錯！

🔁 會變音的結合韻（ㄧㄢ、ㄩㄢ、ㄧㄣ、ㄩㄣ、ㄧㄥ）

某些結合韻會變音或簡化，教學上把它們視為單一學習符號：

  • ㄧㄢ → ㄧ + ㄚㄋ → 變成 ㄧㄝㄋ（ㄚ 變音成 ㄝ）。
  • ㄩㄢ → ㄩ + ㄚㄋ → 變成 ㄩㄝㄋ（ㄚ 變音成 ㄝ）。
  • ㄧㄣ → ㄧ + ㄜㄋ → ㄧㄋ（ㄜ 音不見）。
  • ㄩㄣ → ㄩ + ㄜㄋ → ㄩㄋ（ㄜ 音不見）。
  • ㄧㄥ → ㄧ + ㄜŋ → ㄧŋ（ㄜ 音不見）。

把這些視為整體符號學習，避免拼讀與實際念法不一致。

🔔 三聲字的變調問題（實用提示）

  • 聲 + 或輕聲 → 聲變前半上（例：好聽、打球、買菜、嬸嬸）。
  • 聲 + 聲 → 聲變後半上（例：粉筆、小姐、響板、老鼠）。
  • 聲 + 聲 + 聲（1+2）→ 第一字變前半上、後詞第一字變聲（例：蔣總統、小組長）。
  • 聲 + 聲 + 聲（2+1）→ 前詞組兩個聲字變聲（例：總統府、演講稿）。

用短句練習變調，幫助孩子聽辨並模仿。

👪 給家長的教學小提醒

  • 父母可以用聊天、遊戲的方式隨機進行注音教學，輕鬆有效。
  • 每個孩子有不同發音弱點，可能只有幾個音念不好，家長別操之過急。
  • 一般而言，資質正常的孩子約十週後注音符號能學得很好；重點在練習頻率與趣味性。
